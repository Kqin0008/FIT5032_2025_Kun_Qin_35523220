# SilverCare - 老年人健康关怀平台

一个专为老年人设计的健康关怀和社区资源连接Web应用，基于Vue 3框架开发。

## 🌟 项目简介

SilverCare是一个专注于老年人健康需求的数字化平台，旨在提供简单易用的健康管理服务，连接社区资源，促进老年人的身心健康。

## 🎯 功能特性

### 📱 核心功能
- **健康资讯** - 每日更新的健康小贴士和专业医疗知识
- **活动报名** - 社区健康活动在线报名和管理
- **用户评价** - 真实的用户反馈和平台评分系统
- **个人中心** - 个性化的用户资料和偏好设置

### 🔐 用户认证与授权
- **多角色支持** - 管理员、老年用户、护理人员三种角色
- **安全登录** - 基本认证机制，支持密码加密
- **权限控制** - 不同角色享有不同的访问权限

### 📋 表单验证
- **输入验证** - 多种验证类型（必填、格式、长度等）
- **实时反馈** - 即时显示验证结果和错误提示
- **用户体验** - 友好的错误信息和成功提示

### 💾 数据管理
- **动态数据** - 基于Vue响应式系统的实时数据更新
- **本地存储** - 使用localStorage持久化用户数据和偏好
- **状态管理** - Pinia状态管理确保数据一致性

## 🛠️ 技术栈

### 前端框架
- **Vue 3** - 渐进式JavaScript框架
- **TypeScript** - 类型安全的JavaScript超集
- **Vite** - 快速的构建工具和开发服务器

### UI组件库
- **Bootstrap 5** - 响应式CSS框架
- **Bootstrap Icons** - 丰富的图标库

### 状态管理
- **Pinia** - Vue官方推荐的状态管理库

### 路由
- **Vue Router** - Vue.js官方路由管理器

### 开发工具
- **ESLint** - 代码质量检查工具
- **Prettier** - 代码格式化工具

## 📱 响应式设计

- 使用Bootstrap响应式栅格系统
- 适配桌面、平板、手机等各种设备
- 专为老年用户优化的大字体和高对比度设计

## 🚀 快速开始

### 环境要求
- Node.js 16+
- npm 8+

### 安装和运行

1. **克隆项目**
```bash
git clone <project-url>
cd silvercare-app
```

2. **安装依赖**
```bash
npm install
```

3. **启动开发服务器**
```bash
npm run dev
```

4. **访问应用**
打开浏览器访问 `http://localhost:5173`

### 构建和部署

```bash
# 构建生产版本
npm run build

# 预览构建结果
npm run preview

# 代码检查
npm run lint

# 代码格式化
npm run format
```
